# Archive

This folder keeps old layouts, backups, and prototypes so they do not hide the active app.

Active code is at the repository root:

- `backend/`
- `frontend/`
- `docs/`
- `docker-compose.yml`

Archived folders:

- `legacy-java-backend/`: older Java backend from `03-SourceCodes/backend`.
- `node-prototype-pj_v2/`: Node.js prototype project.
- `merge-final-frontend/`: old frontend merge backup.
- `old-sourcecodes-layout/`: leftover shell from the previous `03-SourceCodes` layout.

Treat archived code as reference material, not the main run path.
